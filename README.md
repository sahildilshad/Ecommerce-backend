# E-Commerce Backend API

This is a backend project built using Node.js, Express.js, and MongoDB.  
It provides REST APIs for user authentication and basic e-commerce features.

##  Features
- User Signup & Login
- JWT Authentication
- Password Hashing (bcrypt)
- User Profile Update
- Product CRUD APIs
- Cart & Wishlist APIs
- Address Management

## Tech Stack
- Node.js
- Express.js
- MongoDB & Mongoose
- JWT
- bcrypt
